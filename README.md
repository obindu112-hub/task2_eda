# Task 2: Exploratory Data Analysis (EDA)

## Objective
The objective of this task is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to understand the data using statistics and visualizations.

## Dataset
- Titanic Dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Steps Performed
1. Loaded the Titanic dataset.
2. Generated summary statistics using `describe()`.
3. Checked for missing values.
4. Created a histogram to visualize the Age distribution.
5. Created a boxplot to identify outliers in the Fare column.
6. Generated a correlation heatmap to understand relationships between numerical features.
7. Created a pairplot to visualize relationships between selected features.

## Key Observations
- The Age distribution shows that most passengers were adults.
- The Fare column contains several outliers.
- The correlation heatmap helps identify relationships between numerical features.
- Pairplots provide a visual comparison of Age, Fare, and Survival.
